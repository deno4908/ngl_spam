## Ngôn Ngữ Lập Trình 

**TOOL:** Python

**Windows**
- Cài packages :  ```pip install -r requirements.txt```
- Run TOOL :  ```python ngl.py```

**Termux**
- Cập nhật hệ thống pkg :  ```pkg update && pkg upgrade```
- Cho phép truy cập file trong máy : ```termux-setup-storage```
- Cài python ```pkg install python```
- Cài packages :  ```pip install -r requirements.txt```
- Run TOOL :  ```python ngl.py```
